## 📁 Organização de Pastas do Projeto

Por terem sido desenvolvidos muitos projetos, organizei-os em pastas. Assim cada pasta contém um projeto diferente e, ao clicar em cada título, você é direcionado para o projeto:

- [**📂 exercicios**](./exercicios) — Pequenos desafios e códigos de fixação para praticar a sintaxe do HTML5 e propriedades do CSS3.

- [**📂 google-search-clone**](./google-search-clone) — Clone de uma página de pesquisa do google, focado em alinhamento centralizado e design minimalista.

- [**📂 hdc-host**](./hdc-host) — Uma landing page completa de uma empresa de hospedagem de sites, estruturando seções de planos e diferenciais.

- [**📂 home-instagram**](./home-instagram) — Recriação da interface de login/home do Instagram, trabalhando com formulários e imagens responsivas para adaptação de mobile e desktop.

- [**📂 site-doceria**](./site-doceria) — Um site comercial para uma doceria, focando em um layout visualmente atraente e organizado.

- [**📂 site-indecor**](./site-indecor) — Landing page para uma empresa de design de interiores, praticando o alinhamento de imagens e textos.

- [**📂 site-menu-hamburger**](./site-menu-hamburger) — Um projeto focado em desenvolver um menu hamburgér utilizando somente HTML e CSS.

- [**📂 site-newsletter**](./site-newsletter) — Criação de um componente de captura de e-mails, focando na estilização de inputs e botões. E utilizo JavaScript para trocar cor do tema do site.

- [**📂 site-pagina-inscricao**](./site-pagina-inscricao) — Página de captura/inscrição para eventos, estruturando bem os blocos de conteúdo e o formulário.

- [**📂 site-pagina-produto**](./site-pagina-produto) — Uma página de vendas de produto, organizando imagens, descrição, preços e botões de conversão com Flexbox.

- [**📂 whatsapp-web-clone**](./whatsapp-web-clone) — Clone da interface do WhatsApp Web, treinando uso de flexbox e float.

## 🚀 Como Executar os Projetos

Como todos os projetos foram desenvolvidos com tecnologias nativas do Front-end, você não precisa instalar nenhuma dependência:

1. Faça o clone deste repositório ou baixe o ZIP.
2. Entre na pasta do projeto que deseja visualizar.
3. Abra o arquivo `index.html` diretamente no seu navegador.
